# Why prompt caching changes the economics of LLM apps

Anthropic's prompt caching reduces cost by 90% on repeated prefixes and improves latency by 80%. Best for: long system prompts, document Q&A, multi-turn agents. Cache lifetime is 5 minutes by default. Architect your prompts so the static parts come first - examples, instructions, retrieved context.
