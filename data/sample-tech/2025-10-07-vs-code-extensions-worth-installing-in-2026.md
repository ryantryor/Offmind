# VS Code extensions worth installing in 2026

GitLens (git history inline), Error Lens (errors in editor), Pretty TypeScript Errors (decode TS hieroglyphics), Better Comments (TODO/HACK highlighting), GitHub Copilot or Cursor integration. Resist installing more - each extension is startup time and a security surface.
