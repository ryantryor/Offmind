# LangChain vs LlamaIndex: when to pick which

LangChain is broader - chains, agents, integrations to everything. Better for production agentic apps. LlamaIndex is RAG-focused with stronger primitives for indexing strategies, query planning, and evaluation. I use LlamaIndex for any project that's primarily about retrieval, LangChain when I need tools and agents.
