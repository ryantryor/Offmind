# Property-based testing finds bugs you'd never write a unit for

Generate random valid inputs, assert invariants. hypothesis (Python), fast-check (JS), QuickCheck (Haskell). Catches off-by-one, encoding, null handling. Especially good for parsers, serializers, math. Cost: tests are slower and harder to debug when they fail. Save for tricky logic.
