# Choosing a license for an open-source project

MIT: maximum permissive, no copyleft. Default for libraries you want adopted. Apache 2.0: like MIT plus patent grant. Better for corporate-friendly projects. AGPL: viral copyleft including over-the-network use. Use to prevent SaaS competitors. Don't pick GPL by accident - it's incompatible with many other licenses.
