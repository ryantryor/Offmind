# Vite is the new default and that's fine

ESM-native dev server, no bundling in dev = instant HMR. Production uses Rollup or now Rolldown. Replaced Webpack/CRA for new projects. Slight ecosystem gaps remain (some webpack plugins lack equivalents). If you're on Webpack and not feeling pain, no rush - but greenfield, Vite.
