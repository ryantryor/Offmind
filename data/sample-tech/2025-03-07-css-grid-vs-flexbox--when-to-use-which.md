# CSS Grid vs Flexbox: when to use which

Flexbox: one dimension at a time (a row or a column). Great for nav bars, card content, centering. Grid: two dimensions simultaneously. Use for page layout, dashboards, image galleries. Don't fight the tool - if you're nesting 3 flexboxes, you probably want grid.
