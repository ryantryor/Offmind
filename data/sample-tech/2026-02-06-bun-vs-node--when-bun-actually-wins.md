# Bun vs Node: when Bun actually wins

Bun is fast: install, test runner, http server. For greenfield JS/TS projects in 2026, it's increasingly the default. Node still wins on: ecosystem maturity, debugging tools, library compatibility (some Node-internal APIs differ). I default to Bun for new projects, Node for projects that need long-term stability.
